## Overview

What you can do:

* Add tasks and set priority and timestamp.
* Delete and complete tasks.

What I learned:

* How to create REST API from scratch
* How to structure the code for React application

## Setup
Download [Node.js](https://nodejs.org/en/download/).
Run this followed commands:

``` bash
# Install dependencies (only the first time)
npm install

# Run the local server
npm run dev

# Build for production in the dist/ directory
npm run build
```
